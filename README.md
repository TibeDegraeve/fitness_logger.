# Fitness Logger
Een programma om trainingen bij te houden.

## Installatie
1. Maak een virtual environment.
2. Installeer requirements: pip install -r requirements.txt
3. Verander src/settings_example.py naar src/settings.py

## Features
- Oefeningen aanmaken (in database)
- Trainingen loggen (sets, reps, gewicht)
- Geschiedenis bekijken (laatste 10 trainingen)
